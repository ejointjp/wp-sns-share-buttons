A WordPress plugin that makes SNS Share and Follow Buttons easily.

== Description ==

SNS share buttons and follow buttons can be installed with simple settings only.

[Plugin page](http://e-joint.jp/works/wp-sns-share-buttons/)

SNSシェアボタンやフォローボタンを簡単な設定だけで設置することができます。

[プラグインページ](http://e-joint.jp/works/wp-sns-share-buttons/)

[Github](https://github.com/ejointjp/wp-sns-share-buttons)

== Installation ==

= Install =

1. Get "WP SNS Share Buttons" from "Plugins > New add" on the WordPress dashboard.
2. Activate this.
3. Set it on the "Settings > WP SNS Share Buttons" in the dashboard.
4. f you want to customize the display, please read the above Plugin Page.

= インストール =

1. 「プラグイン > 新規追加」から「WP SNS Share Buttons」を入手します。
2. プラグインを有効化します｡
3. ダッシュボードの「設定 > WP SNS Share Buttons」で初期設定をします。
4. 表示のカスタマイズをしたい場合は、上記プラグインページをお読みください。
